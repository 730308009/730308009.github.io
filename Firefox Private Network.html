Firefox Private Network是火狐9月初重启的项目

发表于 2020-03-10
  • Yume Shoka
首页 mac

网速 - 这个速度还是可以的
现阶段，此项目限制了地区 IP 下载，但是安装成功后，实际使用没有限制。

一、下载安装火狐浏览器
注意，需要下载美版的火狐浏览器，中国阉割版不行，地址如下：地址（https://www.mozilla.org/en-US/firefox/all/#product-desktop-release）

安装后，是英文版的，大多数人不适应，可以改语言为中文。

（提示：Options - language and appearance - language ）

二、安装插件
项目地址如下：地址 (https://private-network.firefox.com/?utm_source=Blog&utm_medium=AnnouncementPost&utm_campaign=FPN_Beta) 打开后，会出现如下的提示：

This extension is currently available to US desktop users only.（此扩展当前仅对美国桌面用户可用）

仅限美国用户使用
三、安装插件解决方案
第一种： 用美国 IP 进入此网站，那么就可以安装插件，也就是先要 X 墙。（这废话就不说了）

第二种：用别的用户已经下载保存的插件，安装就好了。

插件地址：

链接：https://pan.baidu.com/s/1kKCX6iU2WQs5Bl_RLjXs9Q 提取码：og6q

注意要解压，解压，解压 ！

使用步骤：

一路确定就行了。下载地址错误请留言。

四、注册登录账号
先注册账号，要是以前注册过的，请注意是否在中国版火狐浏览器注册的，和国际版的账号是不互通的，并且此插件仅国际版账号可用。

注册地址（https://accounts.firefox.com/signup?service=sync&context=fx_desktop_v3&entrypoint=menupanel） 注意：国际版 、 国际版，注册完之后可以不用登录 火狐浏览器。

登录插件， 填账号密码就完事了， 还能怎么说。

这插件的使用也是蛮简洁 ， 只有 ON \ OFF 。
